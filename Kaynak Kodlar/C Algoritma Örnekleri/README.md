 - *İşaretçiler (Pointerlar)*
 - *Union Yapısı*
 - *Struct Yapısı*
 - *Dosya ve Klasör İşlemleri (Okuma, Yazma vb.)*
 - *Yinelemeli Fonksiyonlar (Recursive)*
 - *Cursor İşlemleri*
 - *Dizi İşlemleri*
 - *Sıralama Algoritmaları*
 - *Metin İşlemleri*
 - *Tarih İşlemleri*
 - *Matematiksel İşlemler*
   - *Asal Sayılar*
   - *Kare Hesaplama*
   - *Matris İşlemleri*
